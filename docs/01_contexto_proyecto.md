# Contexto del proyecto

Este proyecto organiza el cálculo del **Índice de Transformación Territorial (ITT)** para tres zonas de ciudad:

1. ITT Roosevelt.
2. ITT Avenida Ciudad de Cali.
3. ITT Barrio Obrero.

El objetivo es contar con una estructura sencilla y replicable para calcular el índice por zona, exportar resultados y generar un comparativo general.

## Propósito

Medir el grado de transformación territorial de cada zona mediante una metodología compuesta que integra dimensiones como seguridad, movilidad, entorno urbano, educación/desarrollo y cohesión social.

## Enfoque de trabajo

- Un notebook por zona.
- Una carpeta de datos por zona.
- Una carpeta de resultados por zona.
- Un notebook consolidado para comparar los tres ITT.
- Documentación metodológica.
- Contexto base para un agente de IA.
